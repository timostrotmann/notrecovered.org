# notrecovered.org

Public website for the Not Recovered movement. Built with Astro and Tailwind CSS and includes multilingual pages (EN/DE), a news section, and content pages about PAIS/IACC.

## Structure

- Pages: `src/pages` (localized routes under `src/pages/en` and `src/pages/de`)
- Layouts: `src/layouts`
- Shared UI: `src/components`
- Blog/News content: `src/content/blog` (Markdown/MDX)
- Global styles: `src/styles/global.css`

## Development

- Install: `npm install`
- Run: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`
